# wabe-trans

Mobile application for transportation drivers to manage delivery assignments, enable real-time GPS tracking, start and stop trips, and upload delivery proof documents.

## Support

Thank you for using our application.

If you experience any issues, encounter bugs, or have questions regarding the application, please submit a support request through our **[GitHub Issues page](https://github.com/rabbit-software/wabe-trans/issues)**.

When creating an issue, please include the following information whenever possible:

* **Device model**
* **Operating system version**
* **Application version**
* **Detailed description of the issue**
* **Steps to reproduce the problem**
* **Screenshots or supporting files (if available)**

Our team will review submitted issues and respond as soon as possible.
